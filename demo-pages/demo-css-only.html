Input

Text input with focus ring. Supports placeholder, disabled, and invalid states.

<input data-slot="input" type="text" placeholder="Default input…">
<input data-slot="input" type="email" value="hello@example.com">
<input data-slot="input" disabled placeholder="Disabled">
<input data-slot="input" aria-invalid="true" placeholder="Invalid">

Textarea

Multi-line text input. Same styling tokens as Input.

<textarea data-slot="textarea" placeholder="Write something…" rows="3"></textarea>
<textarea data-slot="textarea" disabled rows="3"></textarea>

Label

Accessible form label. Pairs with inputs via for attribute.

<label data-slot="label" for="my-input">Full Name</label>
<input data-slot="input" type="text" id="my-input">

<!-- Disabled -->
<label data-slot="label" data-disabled="true">Disabled Label</label>

</div>
