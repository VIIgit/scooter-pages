# scooter-pages

Lightweight component demos and pages for Scooter UI.

- Serve locally with a static server (e.g., `python -m http.server 8765`).
- New site pages live under `pages/<name>/index.html` and must use `<main class="scooter-page">`.

See `SCOOTER_PAGES_AGENT_INSTRUCTIONS.md` for contributor guidance.
